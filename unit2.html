Unit 1 : Outcome 2 Review

This website was entirely made in Visual Studio Code. Trello was also used to set up tasks and follow them. While the tools selected to do this project were appropriate, I personally feel that not enough time was allocated for a beginner to research, create and finalize it properly.

The entire website is composed of 6 html files, one css file and images.
